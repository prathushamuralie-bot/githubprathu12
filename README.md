# 📊 COVID-19 Data Analysis Dashboard

## 📌 Project Overview

This project is a COVID-19 data analysis dashboard built using Python.
It helps visualize and understand COVID-19 trends.

## 🚀 Features

* Data analysis using Python
* Interactive dashboard
* Visualization of COVID-19 data

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib / Seaborn
* Streamlit

## ▶️ How to Run

1. Clone the repository
2. Install required libraries
3. Run the Python file:

   ```
   streamlit run streamlit_covid19_dashboard.py
   ```

## 📂 Files Included

* `streamlit_covid19_dashboard.py`
* Dataset files
* Jupyter notebooks
